# API Documentation

## Game records

### GET Record

HTTP: GET
URL: /records/[id]

### WRITE Record

HTTP: PUT
URL: /records

´´´
player1_name - STRING
player2_name - STRING
player1_race - STRING (one of 'Zerg', 'Terran', 'Protoss' or 'Random')
player2_race - STRING (one of 'Zerg', 'Terran', 'Protoss' or 'Random')
map - STRING
winner - INT (1 or 2)
